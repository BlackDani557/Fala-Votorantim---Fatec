## 📖 Sobre o Projeto <a id="sobre-o-projeto"></a>

O **Fala Votorantim** é um Projeto Integrador desenvolvido por estudantes da **Fatec Votorantim**. O objetivo é criar uma solução tecnológica de impacto social que facilite a comunicação entre os munícipes e a Prefeitura.

O sistema permite que cidadãos registrem e acompanhem denúncias urbanas, como buracos, iluminação pública e descarte irregular de lixo, aproximando a população da gestão pública e promovendo transparência.

> ⓘ **Status Atual:** Este repositório contempla o desenvolvimento do **front-end** da aplicação (protótipo funcional da interface e navegação).

---

## ✨ Funcionalidades Implementadas <a id="funcionalidades-implementadas"></a>

- Interface baseada em protótipo do Figma
- Sistema de navegação entre páginas iniciais, usuário e admin
- Simulação de autenticação (usuário e administrador)
- Dashboard administrativo (mock de gestão de denúncias)

## 🚀 Como Executar <a id="como-executar"></a>

### 1. Clone o repositório
```bash
git clone https://github.com/BlackDani557/Fala-Votorantim---Fatec.git
```

### 2. Acesse a pasta do projeto

```bash
cd Fala-Votorantim---Fatec
```

### 3. Abra a aplicação

O ponto de entrada do sistema é:

`Fala Votorantim/Views/HomePage.html`

Você pode abrir de duas formas:

#### 👉 Opção 1: Execução Manual

Navegue pelas pastas do seu sistema operacional até `Fala Votorantim/Views/` e dê um duplo clique no arquivo `HomePage.html`.

#### 👉Opção 2: Extensão Live Server (Recomendado)

1. Abra a pasta raiz do projeto no **VS Code**.
2. Instale a extensão **Live Server**.
3. Abra o arquivo `HomePage.html`.
4. Clique com o botão direito no editor e selecione **"Open with Live Server"**.

> Projeto front-end sem necessidade de instalação de dependências.

## 🎯 Próximas Etapas <a id="proximas-etapas"></a>

Para a evolução do projeto nas próximas etapas acadêmicas, planeja-se:

- Desenvolvimento do ecossistema Back-end.
